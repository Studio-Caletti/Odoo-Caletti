# Odoo-Caletti: Tablero Kanban Personalizado

Este módulo para **Odoo 17** implementa un sistema de gestión de tareas dinámico y visual.

## Características principales:
* **Vista Kanban**: Organización por estados (Nuevo, En Proceso, Finalizado) con colores dinámicos.
* **Automatización**: Al marcar una tarea como 'Hecho', el sistema cambia el color a verde y publica un mensaje automático.
* **Chatter**: Integración con el sistema de mensajería de Odoo para seguimiento de actividades.
* **Reportes PDF**: Generación de informes profesionales que incluyen la identidad visual del tablero.
